# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-03-08

### Added
- Interactive SVG-based Malaysia map with all states
- Pan and zoom functionality
- Custom marker support with popups
- Region click event handlers
- Laravel Blade component integration
- Vanilla JavaScript support
- Customizable CSS styling
- GeoJSON data for all Malaysian states
- Comprehensive documentation and examples
- Demo page with interactive controls

### Features
- VectorMapCore engine for SVG rendering
- Web Mercator projection for accurate mapping
- Responsive design
- Touch and mouse interaction support
- Automatic bounds fitting
- State/region hover effects

### Documentation
- Complete README with usage examples
- Laravel Blade component examples
- Vanilla JavaScript examples
- API reference documentation
